Giới thiệu

Phiên bản Labs Dashboard: mỗi tab là một file HTML riêng trong pages/. index.html mở trực tiếp (file://) vẫn chạy nhờ iframe; khi cần deploy, dùng app.py trên Streamlit.

  • Thêm tab mới: tạo file trong pages/ và thêm route trong assets/app.js.
  • Đổi theme: chỉnh biến CSS trong assets/style.css hoặc .streamlit/config.toml cho bản Streamlit.
  • Triển khai: đẩy repo lên GitHub → Streamlit Community Cloud.

Lưu ý: Ping/trace thực tế cần chạy trên terminal; bản này cung cấp câu lệnh để copy nhanh.
